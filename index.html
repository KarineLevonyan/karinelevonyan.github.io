Research Projects

Domain Engineering for Parametric Face Reconstruction

2022 | Computer Vision & AI

Worked on research on innovative approaches to monocular face reconstruction, developing a novel domain engineering framework that enhances the accuracy and reliability of facial parameter estimation from single images. This work addresses fundamental challenges in computer vision and has practical applications in gaming, virtual reality, and digital content creation.

Evolutionary Computing for Game Design Optimization

2022 | Algorithmic Optimization

Developed an automated puzzle validation system using Graph Genetic Algorithms, revolutionizing the game design process. This research introduced novel heuristic evaluation methods that significantly reduce validation time while ensuring puzzle quality and solvability, directly impacting game development efficiency.

Automated Facial Shape Generation System

2022-2024 | AI & Computer Graphics

With the co-authors, pioneered a system for generating diverse, realistic facial shapes for virtual character models. This work combines advanced machine learning techniques with computer graphics principles to create a robust framework for automated character design, leading to a patent application and practical implementation in virtual environments.
